## What's New in v1.6

### Video Only
- Still photo mode removed — clean video-focused app

### Dual Format Recording
- One shoot saves TWO files simultaneously
- Landscape (wide) — for YouTube / TV / laptop
- Portrait (tall) — for WhatsApp / Instagram Reels / phone
- Toggle Dual Format on/off from top right button

### Zoom
- Tap + / − buttons for zoom
- Pinch gesture on viewfinder
- Say "Zoom in" / "Zoom out" via voice
- Zoom level shown on screen

### Stills While Recording
- Say "Stills" or tap 📷 button during recording
- Screen flashes — photo captured from current frame
- All stills saved along with video

## Voice Commands

| Say            | Action                  | Sound          |
|----------------|-------------------------|----------------|
| "Roll"/"Action"| Start recording         | Triple rising  |
| "Stop"/"Cut"   | Stop recording          | Descending     |
| "Save"         | Save video(s)           | Ping           |
| "Stills"/"Snap"| Photo snap              | Shutter click  |
| "Zoom in"      | Zoom in 0.5x step       | Rising tone    |
| "Zoom out"     | Zoom out 0.5x step      | Falling tone   |
| "Landscape"    | Wide output 90°         | Tick           |
| "Portrait"     | Tall output 0°          | Tick           |
| "Front"        | Front camera            | Pop            |
| "Back"         | Back camera             | Pop            |
| "Grid"         | Toggle grid             | Tick           |
