## What's New in v1.4 — Sound Feedback

All sounds generated by Web Audio API — no external files needed. Works offline.

| Action              | Sound                        |
|---------------------|------------------------------|
| Photo captured      | Camera shutter click         |
| Video record start  | Double beep — high pitch     |
| Video record stop   | Descending tone              |
| Mode switch         | Soft swoosh up               |
| Rotation change     | Soft tick                    |
| Camera flip/mirror  | Soft pop                     |
| Voice activated     | Single ping                  |
| Voice off           | Soft pop                     |
| File saved          | Confirmation ping            |

## Where Files Are Saved
- Photos → Files app → Downloads
- Videos → Files app → Downloads
- From Files app: AirDrop / WhatsApp / Move to Photos
