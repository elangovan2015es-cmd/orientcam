🎙 Voice Commands

Mode
"Photo"
Switch to photo mode
"Video"
Switch to video mode
Capture
"Capture"
Take photo / start recording
"Click"
Take photo
"Record"
Start video recording
"Stop"
Stop recording
Rotation
"Landscape"
Set rotation to 90°
"Normal"
Set rotation to 0°
"Flip"
Set rotation to 180°
"Reverse"
Set rotation to -90°
Camera
"Front"
Switch to front camera
"Back"
Switch to back camera
"Grid"
Toggle grid overlay
